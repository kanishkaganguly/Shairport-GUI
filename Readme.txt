A Swing based GUI application for Shairport (https://github.com/abrasive/shairport) designed to show currently playing song info/metadata on the laptop.

What is Shairport?

This program emulates an AirPort Express for the purpose of streaming music from iTunes and compatible iPods and iPhones. It implements a server for the Apple RAOP protocol.

To set up Shairport, please visit @abrasive's git repo.
